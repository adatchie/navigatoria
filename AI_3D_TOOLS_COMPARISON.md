## 1. Tripo AI

### 概要
VAST AI Research製。「最低摩擦で使えるAI 3D生成」を旗印に、MCP公式対応、
Blender直接統合、Unreal/Unity対応など開発者フレンドリーな設計。

### 品質
| 評価項目 | スコア | 詳細 |
|----------|-------|------|
| ジオメトリ精度 | 8/10 | 輪郭・シルエットの再現は高精度 |
| テクスチャ品質 | 8/10 | PBRテクスチャ自動生成、Ultra HD対応 |
| **トポロジー** | **9/10** | **クワッドメッシュ優先。ゲームエンジン直接利用可** |
| 生成速度 | 8/10 | 平均30〜60秒 |
| 一貫性 | 8/10 | 同プロンプトでの再現性が高い方 |

```
入力形式:
  - テキスト → 3D (text_to_model)

  - GLB / GLTF / FBX / OBJ / STL / USDZ

追加機能:
  - Ultra HDテクスチャ
  - AIリギング
  - アニメーション付与
  - Lock Frame (特定フレームを固定した生成)
  - モデルライブラリ (大量の生成済みアセット再利用可能)
```

## 2. Meshy AI

### 概要
ゲーム開発者に最も広く使われているAI 3D生成ツールの一つ。
Meshy 5（2025年7月）→ Meshy 6（2025年後半）で品質が大幅向上。
「テキスト/画像から数分でゲームレディなアセット」が強み。

### 品質（Meshy 6基準）
| 評価項目 | スコア | 詳細 |
|----------|-------|------|
| ジオメトリ精度 | 9/10 | Meshy 6でエッジの鋭さ・構造の正確さが大幅改善 |
| テクスチャ品質 | 9/10 | PBRテクスチャ品質は全ツール中トップクラス |
| **トポロジー** | **8/10** | **整理されたメッシュ。慎重なプロンプトで更に改善** |
| 生成速度 | 7/10 | 平均60〜90秒（Tripo比でやや遅い） |
| 一貫性 | 9/10 | 高い再現性・安定性 |

**Meshy 6の主要改善点:**
- キャラクター・有機系: 豊かな表面ディテール、正確な解剖学、表情
- **ハードサーフェス（帆船はこれ）**: エッジの鋭さ向上、クリーンな構造
- 生成→クリーンアップ時間が「数日→数分」に短縮

### 機能詳細
```
入力形式:
  - テキスト → 3D
  - 画像 → 3D（テクスチャあり/なし選択可）
  - AIテクスチャリング（既存モデルへのテクスチャ付与）

出力フォーマット:
  - GLB / FBX / OBJ / USDZ / STL / ABC

追加機能:
  - リメッシュ（ポリゴン数最適化）
  - 自動リギング（94%成功率）
  - アニメーション生成
  - Webhook対応（非同期処理）

クレジット消費（API）:
  - Text to 3D (Meshy 6): 20クレジット
  - Image to 3D (テクスチャあり): 15〜30クレジット
  - リメッシュ: 5〜10クレジット
  - 自動リギング: 5クレジット
```

## 3. Rodin (Hyper3D)

### 概要
DeemosTech（旧DeemosTech）が開発。「映画品質の3D生成」をターゲットに、
学術研究（SIGGRAPH 2024・2025で発表）をベースにした高精度モデル。
Gen-2は100億パラメータモデル。

### 品質（Rodin Gen-2基準）
| 評価項目 | スコア | 詳細 |
|----------|-------|------|
| ジオメトリ精度 | 10/10 | スキャン品質のリアリズム。細部の彫刻レベル |
| テクスチャ品質 | 10/10 | 4K PBRテクスチャ。フォトリアリスティック |
| **トポロジー** | **10/10** | **クワッドメッシュ。アニメ・レンダリングパイプライン直接利用可** |
| 生成速度 | 6/10 | 平均85〜120秒 |
| 一貫性 | 6/10 | 同プロンプトで結果が変わりやすい |

**技術的特徴:**
- CLAY モデル: 15億パラメータ（Multi-resolution VAE + Latent Diffusion Transformer）
- Gen-2: 100億パラメータへスケールアップ
- 18K/50K のクワッドメッシュ出力
- T/A Pose 強制（キャラクター向け）
- NeRF ↔ メッシュ変換

### Claude MCP連携
```
MCPサーバー: DeemosTech/blender-mcp-rodin-integration
状態: 公式が出しているBlender + Rodin統合（非公式blender-mcpと連携）

概要:
  - blender-mcp 経由でClaude ↔ Blender ↔ Rodin の3者連携
  - Rodinの無料試用キーでBlender内から生成可能
  - 「Poly Havenとの統合」でHDRIや環境アセットも同時取得

セットアップ:
  1. blender-mcp addon インストール
  2. Claude Desktop の config に追加
  3. Blenderで「Connect to Claude」
  4. 「Hyper3D Rodinで帆船モデル生成して」と指示するだけ
```

**注意:** Rodinの無料キーは1日の生成上限あり。大量生成はAPI有料キーが必要。

| プラン | 月額 | クレジット/月 | 1モデルコスト | 商用利用 |
|--------|------|-------------|-------------|---------|
| Free | $0 | 限定 | — | ✓ (制限あり) |
| Creator | $24〜30 | 30クレジット | **約$0.75〜1.00** | ✓ |
| Business | $120 | 60クレジット | **約$2.00** | ✓ |

⚠️ **コスト注意**: 他ツールの4〜10倍のコスト。20船種 × 2回 = $16〜40（fal.ai経由）

### 強み
- **品質は全ツール中ナンバーワン** — フォトリアル・映画品質
- **クワッドメッシュ品質が最高** — リトポ作業ゼロ
- Blender MCPとの統合が公式サポート（DeemosTechが提供）
- 無料キーで毎日少数を試せる
- 学術研究ベースで継続的な品質向上

### 弱み
- **コストが最高** → 大量生成には不向き
- 生成の一貫性がやや低い（同プロンプトで結果が変わる）

- MCPサーバーが公式ではなくblender-mcp依存

## 4. SEELE

### 概要
「AIでゲームを作る」ためのオールインワンプラットフォーム。
3Dモデル生成ツールではなく、**ゲームエンジン代替**が本質。
テキストプロンプトから2D/3Dゲームを数分で完成させる「ノーコードゲームメーカー」。

### 品質
| 評価項目 | スコア | 詳細 |
|----------|-------|------|
| ジオメトリ精度 | 7/10 | 汎用的な品質。特化した精度はない |
| テクスチャ品質 | 7/10 | ゲームプレイに支障ない程度 |
| トポロジー | 7/10 | 自動リギング成功率94%（人型・四足獣） |
| 生成速度 | 9/10 | 30〜60秒（最速クラス） |
| 一貫性 | 8/10 | 安定した出力 |

### 主な機能
```
3D/2Dアセット生成:
  - テキスト→3Dモデル（30〜60秒）
  - スプライトシート生成（5〜10秒）
  - PBRテクスチャ生成
  - 5M+のアニメーションライブラリ

エクスポート対応:
  - Unity (.unitypackage / C# export)
  - Three.js (WebGL/browser code export)  ← 我々のプロジェクトと一致
  - FBX / GLTF / OBJ / PNG / JPG

ゲームエンジン統合:
  - 物理エンジン内蔵
  - ゲームロジック生成
  - ゲーム全体を会話で作成可能
```

### Claude MCP連携
```
MCPサーバー: 確認できず（未対応）
API: 一般公開されていない
Claude連携: 非対応

→ MCPによる自動化パイプラインには組み込めない
```

### 料金
詳細非公開。AITopToolsでの評価は5.0/5（レビュー数少）

### 判定: **本プロジェクトには不向き**
SEELEは「ゲームを丸ごとAIで作る」ためのツールであり、我々が求める
「帆船の高品質3Dモデルを取り出してThree.jsに統合する」という用途には合わない。
Three.jsへの書き出しは対応しているが、モデルのパーツ分離・カスタマイズ性が不足する。

## 8. 最終推奨

### 本プロジェクト（帆船ゲーム）への推奨構成

```
メイン: Tripo AI Professional ($11.94/月)
  理由:
  ① 公式MCPでモジョンさんと同じ自動化が即実現
  ② クワッドメッシュでゲームエンジン直行
  ③ コスト最安（20船種を$5〜15で生成可能）
  ④ Blender公式Addonで後処理自動化
  ⑤ 多視点入力で帆船の特定のアングルを指定して精度向上

必要に応じて: Meshy AI Pro ($20/月) を1ヶ月だけ追加
  理由:
  ① Meshy 6でハードサーフェス品質が特に向上
  ② 主要5船種（ガレオン、ガレー等）の高品質版作成
  ③ リメッシュ機能でポリゴン数を自動最適化

特別用途: Rodin は blender-mcp の無料キーで試し生成のみ
  理由:
  ① 無料キーで毎日数本は生成できる
  ② タイトル画面用の超高品質版を少数だけ生成
  ③ コストを抑えながら最高品質を一部に投入

SEELE: 除外
  理由: 目的が「ゲームを丸ごと作る」ツールであり、
        本プロジェクトの「モデル抽出・Three.js統合」には不向き
```

**合計コスト目安:** $12〜32/月 × 1〜2ヶ月 = **$12〜64 で全船種完成**
